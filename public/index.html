公式演绎

牛顿第二定律

F：力 (N)

m：质量 (kg)

a：加速度 (m/s²)

推导过程：

  1. 加速度定义：a = Δv/Δt
  2. 速度公式：v = v₀ + at
  3. 位移公式：s = v₀t + ½at²
  4. 联立求解得：F = ma

动能定理

W：合外力做功 (J)

Eₖ：动能变化 (J)

v：速度 (m/s)

推导过程：

  1. 功的定义：W = F·s = ma·s
  2. 位移关系：v² - v₀² = 2as
  3. 消去加速度：W = ½mv² - ½mv₀²
  4. 即 W = ΔEₖ
