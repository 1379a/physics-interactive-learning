关于牛顿运动定律，下列说法正确的是

简单力学2024年高考全国甲卷
A.牛顿第一定律描述了物体在不受外力作用时的运动状态✓
B.牛顿第二定律适用于惯性参考系和非惯性参考系

15.

两个质量分别为m₁和m₂的物体发生完全弹性碰撞，碰撞前m₂静止。若碰撞后m₁反向运动，则

中等力学2020年高考全国卷II
A.m₁ > m₂
B.m₁ = m₂
C.m₁ < m₂✓
D.无法确定
正确答案：C

解析：

完全弹性碰撞中，动量守恒和动能守恒： m₁v₁ = m₁v₁' + m₂v₂'（动量守恒） ½m₁v₁² = ½m₁v₁'² + ½m₂v₂'²（动能守恒） 已知碰撞前m₂静止，即v₂₀ = 0。解此方程组可得： v₁' = (m₁-m₂)v₁/(m₁+m₂) v₂' = 2m₁v₁/(m₁+m₂) 碰撞后m₁反向运动，即v₁' < 0（以v₁方向为正方向）。 要使v₁' < 0，需要(m₁-m₂)/(m₁+m₂) < 0 由于m₁+m₂ > 0，所以需要m₁ - m₂ < 0，即m₁ < m₂。 物理意义： - 当m₁ = m₂时，v₁' = 0，m₁停止，m₂以v₁速度前进 - 当m₁ > m₂时，v₁' > 0，m₁继续向前运动 - 当m₁ < m₂时，v₁' < 0，m₁反向运动 因此选C：m₁ < m₂
相关知识点：
弹性碰撞动量守恒动能守恒

在同一均匀介质中有两个振幅相等、频率为100 Hz的相干波源A与B，A处于波峰时，B处于波谷。波A与波B相距30 m，波速为400 m/s，则在A、B连线上距A波源13 m的点将

A.振动加强
B.振动减弱✓
C.既不加强也不减弱
D.无法确定
正确答案：B

解析：

波长λ = v/f = 400/100 = 4 m A处于波峰时，B处于波谷，说明两波源的相位差为π（反相）。 P点距A为13 m，距B为30-13=17 m。 波程差Δr = |r_B - r_A| = |17 - 13| = 4 m = 1λ 考虑相位差：两波源初始相位差为π，波程差为整数倍波长。 相位差 = π + 2π(Δr/λ) = π + 2π = 3π = π（模2π） 由于相位差为π（反相），所以P点振动减弱。 或者用另一种方法： 由于A、B反相，振动加强条件变为Δr = (n+1/2)λ 振动减弱条件变为Δr = nλ Δr = 4 m = 1λ，满足减弱条件。 所以选B：振动减弱。
